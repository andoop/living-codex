# 命令参考 · Commands

> 这些是发给 **AI** 的命令，不是 shell 命令。装好 skill、重启 agent 后：
> - **Cursor / Claude Code / Kiro**：直接把 `narrate …` 当普通消息发给 AI（或用自然语言说"给这个项目做业务叙事 / 剥一条 XX 旅程"）。
> - **Codex CLI**：先 `$narrator`，再用自然语言说。
> - 安装：`curl -fsSL https://raw.githubusercontent.com/andoop/living-codex/main/install.sh | bash`

---

## Living Codex 做什么

把陌生代码库**按业务旅程**讲成一本书，一层层往里讲：

```
L0 这是什么产品 → L1 有哪几块能力 → L2 一条旅程怎么走
→ L3 有哪些分支和规则 → L4 底层怎么实现 → L5 对应哪段代码
```

读起来像产品文档。产物落在项目的 `docs/codebook/narrative/`，是纯 Markdown，Obsidian 打开就有图谱、反链、搜索。

> **一句话提醒**：所有覆盖率、机械校验，证明的都是"清单列全了"，**不等于**"业务没遗漏、已验证正确"。产物是一份**诚实的导览草稿**：帮你快速理解、把你带到该读的代码、并主动告诉你哪些它没覆盖。业务对不对，仍要你对着代码自己核。详见 [honesty.md](honesty.md)。

## 命令清单

### `narrate [path]` — 把项目讲成业务书
从业务入口出发，梳理所有旅程，产出 L0–L5 六层导览书。

| 参数 | 说明 | 默认 |
|---|---|---|
| `[path]` | 范围，可只指一个子目录 | 项目根 |
| `--journey <名>` | **只剥一条指定旅程**（首次推荐，跑得快） | — |
| `--resume` | 中断后接着跑 | — |
| `--backend auto\|codegraph\|repomix\|none` | 可选的解析器后端（见下） | auto（没有就降到 none） |
| `--lang zh\|en\|...` | 产物语言 | 跟当前对话 |

```text
narrate .                       # 梳理全项目的业务旅程
narrate --journey 工程下载       # 只剥一条旅程（首次推荐）
narrate . --backend codegraph   # 叠加静态符号图，让静态结论更可信
```

> **`narrate .` 是个重活**：它要派一堆子 agent、列入口、多角色追踪、配连边、独立审核，前几十秒甚至几分钟看不到产物是正常的。**首次强烈建议先用 `--journey` 锁一条旅程**，看清成品长啥样，再决定要不要铺全量。

它内部的工作流程是：

```
定范围 → 列入口（含状态触发器）→ 三视角追踪 → 配连边
→ 调和分歧 → 归并成六层 → 写书 → 独立审核 → 机械校验通过才算完成
```

### `narrate --audit` — 单独再审一轮
对已经写好的书，另派一个**独立 agent**（跟写书的不是同一个）来挑刺：抽查引用是不是真的、行为有没有被乱标"已确认"、连边有没有漏配、是不是用一个高覆盖率数字盖住了没追的子系统……报告落在 `docs/codebook/narrative/audit/`。
> `narrate` 默认就含这一步；`--audit` 是事后单独再跑一次。

### `narrate --lint [--since <git-ref>]` — 增量保鲜
代码改了之后，找出 `git diff` 命中的文件，把用到这些文件的旅程标上"⚠️ 待复核（可能过期）"。**只提醒，不阻断、不自动改。**
> 注意：grep 还能搜到这个符号，**不代表**这段叙事没过期（符号还在、含义可能变了），所以它只敢说"待复核"。

## 后端 `--backend`（可选，不是必需）

| 值 | 作用 | 对可信度的影响 |
|----|------|----------------|
| `none`（默认可降到这） | 纯 AI 读文件 | 一切连边、运行期结论都只标「推断」 |
| `codegraph` | 借静态符号图 | 静态结构结论可升到「已确认(解析器)」；运行期仍「推断」 |
| `repomix` | 借 [repomix](https://repomix.com) 预打包上下文 | 更快更省 token，可信度档不变 |
| `auto` | 有就用，没有就降到 `none` | 自适应 |

**核心永远零运行时、可移植**；后端缺了自动降级，不影响结构。

---

## 三档可信度（产物里随处可见）

- **已确认** — 这个符号在文件里**确实出现过**（grep 核对过）。仅此而已，不代表行为为真。
- **推断** — 没跑过、靠读代码得出的判断。一切"调用 / 顺序 / 触发 / 运行期"结论封顶到这一档。
- **未解之谜** — 得跑起来、或需要更多上下文、或没读到；会说明该去读哪个文件。

完整规则见 [honesty.md](honesty.md)。
