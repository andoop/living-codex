# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) v2.1.

In short: be respectful, assume good faith, no harassment or discrimination. Maintainers may remove, edit, or reject contributions and comments that violate this code.

Report unacceptable behavior to the maintainers via a private issue or the contact in [SECURITY.md](SECURITY.md). All reports are handled confidentially.
